# 车友荟 - 国内服务器部署指南

## 一、前置准备

### 1. 购买云服务器
- **推荐配置**：2核4G（最低）/ 4核8G（推荐）
- **推荐厂商**：阿里云 / 腾讯云 / 华为云
- **系统**：Ubuntu 22.04 LTS
- **带宽**：5Mbps起（含图片资源）
- **地域**：华东/华南（根据目标用户选择）

### 2. 域名 + ICP备案
- 购买域名（如 cheyou.com.cn.cn）
- **ICP备案必须完成才能绑定国内服务器80/443端口**
- 备案周期约7-20个工作日，建议尽早提交
- 备案期间可先用IP:3000临时访问

### 3. SSL证书
- 云厂商提供免费DV证书（1年）
- 或使用 Let's Encrypt（免费，自动续期）

## 二、服务器环境搭建

### 1. 基础环境
```bash
# 更新系统
apt update && apt upgrade -y

# 安装Node.js 20
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt install -y nodejs

# 安装PostgreSQL 16
apt install -y postgresql postgresql-contrib

# 安装Nginx
apt install -y nginx

# 安装PM2
npm install -g pm2

# 安装Git
apt install -y git
```

### 2. 配置PostgreSQL
```bash
# 切换postgres用户
sudo -u postgres psql

# 创建数据库和用户
CREATE USER cheyou WITH PASSWORD '你的强密码' CREATEDB;
CREATE DATABASE cheyou OWNER cheyou;
GRANT ALL PRIVILEGES ON DATABASE cheyou TO cheyou;
\q
```

### 3. 配置防火墙
```bash
ufw allow 22     # SSH
ufw allow 80     # HTTP
ufw allow 443    # HTTPS
ufw enable
```

## 三、项目部署

### 1. 部署项目代码
```bash
# 创建项目目录
mkdir -p /var/www
cd /var/www

# 从GitHub拉取代码（需先配置SSH key或使用HTTPS+token）
git clone https://github.com/你的用户名/cheyou.git
cd cheyou

# 安装依赖
npm ci

# 配置环境变量
cp .env.production .env
# 编辑 .env 填入生产数据库密码和域名
nano .env

# 生成Prisma Client
npx prisma generate

# 执行数据库迁移
npx prisma migrate deploy

# 导入种子数据（首次部署）
npx tsx prisma/seed.ts

# 构建
npm run build

# 启动服务
pm2 start ecosystem.config.js
pm2 save
pm2 startup  # 开机自启
```

### 2. 配置Nginx
```bash
# 复制配置文件
cp nginx.conf /etc/nginx/sites-available/cheyou
ln -s /etc/nginx/sites-available/cheyou /etc/nginx/sites-enabled/

# 编辑配置，替换域名
nano /etc/nginx/sites-available/cheyou
# 把 your-domain.com 改为实际域名

# 配置SSL证书
mkdir -p /etc/nginx/ssl
# 上传证书文件到 /etc/nginx/ssl/cheyou.pem 和 cheyou.key

# 测试并重启
nginx -t
systemctl restart nginx
```

### 3. 后续更新部署
```bash
cd /var/www/cheyou
bash deploy.sh
```
